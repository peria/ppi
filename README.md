# ppi
ppi は円周率の計算をするプログラムを書いていくお一人様プロジェクトです。十分なストレージがあれば現実的な時間で１兆桁とか計算できるようになることを目標にしています。なので円周率の計算で必要になる多倍長計算に [GMP](https://gmplib.org/)のようなライブラリは用いません。
